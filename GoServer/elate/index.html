API

A guide to the Request and Response parameters. (Phase 1)

GetNotificationsByUserID

Method: GET | URL: /get_notifications_by_user_id
To retrieve all notifications belong to a single user.

Request

Parameter Type Required
user_id uint Yes
limit uint No. Default:0, Max:50

Response

Parameter Type
RespMeta RespMeta
notification_id uint
notification_text string

Results are sorted in descending order.

Tech Stack

These are the tech stacks we used for this project.

JavaScript

HTML, CSS and JavaScript to build the front end UI interface.

Go Language

Go Language to build the API server backend.

REST

APIs are HTTP powered and conforms to the RESTful architecture.

MySQL

Database Management System is implemented using MySQL.

Docker

The API server is containerized with Docker for better CI/CD and ease of deployment.

Heroku

The APIs are deployed publicly, powered by Heroku Container Registry.
